# 契約管理システム

Salesforce 等のライセンス契約を管理するローカルアプリです。インストール不要・ブラウザだけで動作します。

## 使い方

`index.html` をダブルクリックしてブラウザで開くだけです。（推奨: Chrome / Edge）

初回は左下の **サンプル投入** を押すと、動作確認用のデータが入ります。

## 機能

| 画面 | 内容 |
|------|------|
| ダッシュボード | 契約数・有効・更新間近・期限切れ・金額合計の KPI、対応が必要な契約、担当営業別サマリ |
| 契約一覧 | 検索・状態フィルタ・担当営業フィルタ・列ソート・追加/編集/削除/詳細 |
| 企業一覧 | 企業ごとの契約数・金額・要対応件数、追加/編集/削除 |
| 更新アラート | 期限切れ / 30日以内 / 31〜60日以内 の契約を抽出 |

- 管理項目: 企業・部署・ライセンス種別・数量・単価・金額・契約期間・自動更新・**担当営業**・備考
- 終了日 60 日前から「更新間近」、過ぎると「期限切れ」を自動判定
- CSV エクスポート / インポート（左下ボタン）
- データはブラウザの localStorage に保存（このPC・このブラウザ内に保持）

## データの保存場所

ブラウザの localStorage に保存されます。バックアップは **エクスポート** で CSV を保存してください。
別PCへ移す・Web版へ移行する際もこの CSV を **インポート** できます。

## 開発 / テスト

```bash
npm install      # jsdom (テスト用) を導入
npm test         # 単体テスト + 動的(DOM)テスト
npm run test:unit
npm run test:dom
```

- `app.js` の純粋ロジックは `core` namespace に集約し、Node で単体テスト（27件）
- `test/dom.test.js` は jsdom で実画面を操作する動的テスト（11件）
- 静的解析: `node --check app.js`

## Web 化に向けて

`core`（ロジック）と DOM 層を分離済みです。Web 版では localStorage の `db.load/save` をサーバ API に差し替えるだけで移行できます。
